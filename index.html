There are many ways to learn this but you will never be able to apply the syntax untill you practice. I have listed a few suggestions below but make sure that once you start you stick with it.

There are many different elements involved in HTML but once you learn the basics of the syntax, it becomes much easier to read code. Of course the best way to learn coding is to simply start coding.

CSS

The most important thing to remeber about CSS code is that each selector is used to target the specified class. It took me a while to learn how different elements effect the class or id and that comes with practice. The syntax isn't difficult, but remembering all of the properties and thier effect can be tricky, especially paddding, margin, content, and border. These three can seem interchangable but they are much more complex than you may thnk; for example:


#navbar {
background-color: lightgrey;
flex-direction: column;
width: 200px;
min-width: fit-content;
height: 100%;
position: fixed;
color: darkslategrey;
border-right: 2px solid darkslategray
}

This code is what I used to make the navigation bar on the left (there is actually more to get the exact design you see but you get the point). These properties take a while to grasp but once it clicks it becomes so much easier. Here is a simple explination of each property:

  • Content – The content of the box, where text and images appear
  • Padding – Clears an area around the content. The padding is transparent
  • Border – A border that goes around the padding and content
  • Margin – Clears an area outside the border. The margin is transparent

I have found ist easier to start my CSS stylesheets with this code:

/* Reset styles */
body, h1, h2, h3, h4, h5, h6, p, ul, ol, li, figure, figcaption, blockquote, dl, dd {
margin: 0;
padding: 0;
}

This resets all of the margin and padding to zero and allows you to start from scratch.
